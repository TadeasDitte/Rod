# Rod
A family tree building web where you log not just dates and names but also history
Btw, [Rod](https://en.wikipedia.org/wiki/Rod_(Slavic_religion)) is the god of family and ancestors, but it also means family lineage in Slovak so double entendre, yay.

## Core Idea

Most of these platforms either suck, sell your data or they are not very feature rich. When i had to do this for a history class i ended up using a mermaid graph anyway because it somehow worked better then dedicated webs. So i decided to make this thingi thing.

## Planned features

Platform features:
- Link historical documents to family members
- Merge trees between users
- The usual [birth, death, relationship]
QoL:
- SSO by google, gh, discord
- Internationalisation
